How to add a project from template 

by Shunguang Wu 
(3/26/2023)

-----------------------------------
1. for static lib: create libXyz
   - copy libTemplate/ into libXyz/
   - go into libXyz/
     then rename libX.* to libXyz.*

2. for exe: create appXyz
   - copy exeTemplate/ into appXyz/
   - go into appXyz/
     then rename xApp.* to appXyz.*
   - in VS add project appXyz, then add your files
